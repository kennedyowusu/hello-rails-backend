# ğŸ“– Hello-Rails-Backend <a name="about-project"></a>

Hello Rails Backend is an in-depth tutorial that guides you through the process of connecting a React front-end application to a Ruby on Rails backend using an API.The backend, built with Rails, serves as the API and holds the database that the frontend can interact with.

## ğŸ›  Built With <a name="built-with"></a>

This app is build with Ruby on Rails and Postgresql.

### Tech Stack <a name="tech-stack"></a>

> <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
> <li><a href="https://rubyonrails.org/">Ruby on Rails</a></li>
> <li><a href="https://www.postgresql.org/">Postgresql</a></li>

### Key Features <a name="key-features"></a>

> - A random greeting is displayed on the frontend app every time a request is made to the Api

## To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Mac or PC
- Install Ruby
- Install Rails
- Install Postgresql
- Install npm
- Understanding of Ruby

### Setup

- Clone this repository to your desired folder:

```sh
 git clone https://github.com/felixDev22/hello-rails-back-end
```

- Navigate into the cloned folder

```sh
 cd hello-rails-react

```

### Install

- Install this project with:

```sh
 bundle install
 yarn install
```

- Set up the database
  Change the username and password of your postgres account in `config/database.yml`

```sh
 rails db:create db:migrate db:seed
```

- Start the server

```sh
 ./bin/dev
```
